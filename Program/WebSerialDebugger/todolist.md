# ToDoList

## ✅ 已完成
- [x] 1. 可以选择发送的数据文件，如D:\data.txt
- [x] 2. 按字节发送，可以选择每个字节之间的时间间隔ms，1-1000ms
- [x] 3. 可选发送模式(以数据为1,2,3为例)
    - [x] 3.1 连续模式1 2 3 1 2 3...
    - [x] 3.2 oneshot模式，1 2 3发完就停止 
    - [x] 3.3 回环模式1 2 3 2 1 2...
- [x] 4. 可以选择发送数据的格式，ASCII或Hex
    - [x] 4.1 如果是ASCII，则发送数据文件中的内容,如果读取到",",";"则跳过，认为是分隔符
    - [x] 4.2 如果是Hex，则发送数据文件中的内容，并将每个字节转换为16进制字符串，如0x01 0x02 0x03

## 🔄 新功能
- [x] 5. Pause/Continue 功能：Stop 后可以从停止位置继续发送
